## Anatomy

1. Parent item: An item that expands to reveal child items.
2. End item: An item without children.
3. Level indicator lines: A visual indicator representing the depth of an item.
4. Toggle: A toggle that expands or collapses a parent item.
5. Leading visual (optional): A visual cue for additional context.
6. Item label: The text label for an item.
7. Trailing visual (optional): Same as a leading visual, but on the opposite side.
